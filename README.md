# Cloud-Native DevOps Learning Project

This was an beginner-level DevOps project created to understand the basics of containerization,
CI/CD pipelines, and Kubernetes deployment using a simple Python application.

## Project Overview
- Built a simple Python application
- Containerized the application using Docker
- Created a basic CI pipeline using GitHub Actions
- Deployed the containerized application on local Kubernetes (Minikube)
- Used Git and GitHub for version control

## Tools & Technologies
- Python
- Docker
- GitHub Actions
- Kubernetes (Minikube)
- Git & GitHub

## What I Learned
- How Dockers images and its containers work
- Basics of CI/CD pipelines and automation
- Kubernetes concepts like Pods, Services, and Deployments
- Managing source code using Git
